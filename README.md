📊 Sales Report Dashboard (Power BI)
📌 Overview

This project presents a Sales Report Dashboard built using Power BI.
It provides insights into sales performance, revenue trends, and key business metrics to support data-driven decision-making.

🎯 Objectives
Analyze overall sales performance
Track revenue and profit trends
Identify top-performing products and regions
Monitor key business KPIs
📂 Dataset

The dataset includes:

Sales transactions
Product details
Customer information
Regional data

📊 Dashboard Features
📈 Sales trends over time
🌍 Region-wise sales analysis
🏆 Top products and categories
💰 Profit and revenue breakdown
🔍 Interactive filters and slicers
🛠️ Tools & Technologies
Power BI
CSV (data source)
Data cleaning & transformation using Power Query

🚀 How to Use
Open the .pbix file in Power BI Desktop
Refresh the dataset (if needed)
Use filters/slicers to explore insights
